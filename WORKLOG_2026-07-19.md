## 알아둘 점

- 배포는 GitHub Actions로 자동 진행되며 보통 1~2분 내 완료된다. 화면에 반영이 늦어 보이는 경우 대부분 **브라우저/기기 캐시** 문제이며, 시크릿 창으로 확인하면 최신 버전인지 바로 확인 가능하다.

- 민법·경영학은 아직 기출 분석 미완료 상태로, 챕터 목록만 노출되고 있다 (`data/topic_stats.dart`).
